项目根目录
├── cache/          # 缓存目录，存放临时截图、识别缓存、中间临时文件
├── config/         # 配置目录，存放ADB连接参数、识图模板配置、任务规则配置
├── data/           # 数据目录，存放任务运行日志、操作记录、历史执行数据
├── debug/          # 调试目录，存放调试日志、截图对比文件、错误排查数据
├── externals/      # 外部依赖目录，存放ADB工具包、识别模型、第三方依赖组件
├── Python/         # Python核心源码目录
│   ├── adb/        # ADB连接与设备操作模块
│   ├── ocr/        # 图像识别、模板匹配核心模块
│   ├── task/       # 自动化任务调度与执行模块
│   └── main.py     # 项目主程序入口
├── resource/       # 资源目录，存放识图模板图片、静态资源、UI素材文件
├── DependencySetup_依赖库安装.bat  # Windows环境一键依赖安装脚本
└── 主程序启动入口.exe  # 打包后的可执行程序入口（可选）
